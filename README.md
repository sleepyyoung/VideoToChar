# VideoToChar

## 利用python实现的视频字符化小程序

涉及到的第三方库有：

```
opencv`，`PIL`，`moviepy`，`theading
```

没有安装的请自行pip安装

main函数三个参数分别为(原始视频路径,字符视频名称,最终视频名称)

其中第二个参数字符视频名称为中间过程产品，导出的视频是没有声音的
